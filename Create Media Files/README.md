# Image Prompt Helper 🐕

**Скилл для улучшения промптов и выбора лучших изображений для проекта "Собачья мудрость"**

---

## 📖 Описание

Этот скилл помогает создавать качественные изображения через платформу [arena.ai](https://arena.ai/ru/image):
- Улучшает короткие промпты до детализированных описаний
- Сохраняет промпты в файлы
- Анализирует и сравнивает сгенерированные изображения
- Рекомендует лучший вариант с обоснованием

---

## 🚀 Быстрый старт

### 1. Напиши короткий промпт

Просто опиши, что хочешь увидеть:
```
Две собаки дружат
```
```
Щенок лабрадора играет в саду
```
```
Немецкая овчарка на закате
```

### 2. Получи улучшенный промпт

Qwen Code превратит твой запрос в детальное описание на английском языке с:
- Конкретными персонажами и их внешностью
- Окружением и освещением
- Художественным стилем
- Настроением и атмосферой

### 3. Одобри промпт

Проверь улучшенный промпт. Если нужно — попроси внести правки.

### 4. Сгенерируй изображения

- Qwen Code откроет ссылку: https://arena.ai/ru/image
- Скопируй промпт в поле на сайте
- Получи 2 изображения от разных моделей

### 5. Загрузи изображения в чат

Перетащи оба изображения в чат Qwen Code.

### 6. Получи анализ и рекомендацию

Qwen Code сравнит изображения по 6 критериям:
1. Соответствие промпту
2. Композиция
3. Цветовая палитра
4. Детализация
5. Стиль
6. Эмоциональный отклик

И порекомендует лучшее с подробным обоснованием.

### 7. Сохрани результат

Скачай лучшее изображение в папку `results/`

## 🎨 Критерии анализа изображений

| № | Критерий | Что оценивается |
|---|----------|-----------------|
| 1 | **Соответствие промпту** | Все ли элементы промпта переданы точно |
| 2 | **Композиция** | Баланс, фокус, расположение объектов |
| 3 | **Цветовая палитра** | Гармония цветов, настроение |
| 4 | **Детализация** | Проработка текстур и мелких элементов |
| 5 | **Стиль** | Художественная целостность |
| 6 | **Эмоциональный отклик** | Насколько изображение "цепляет" |

## 🐕 Особенности для проекта "Собачья мудрость"

### Предпочтительные цвета:
- ✅ Древесные оттенки
- ✅ Травянистые зелёные
- ✅ Лиловые и фиолетовые
- ✅ Тёплые персиковые и оранжевые

### Избегать:
- ❌ Холодные офисные цвета (серый, стальной)
- ❌ Кислотные оттенки
- ❌ Современный минимализм

### Персонажи:

**Смельчак:**
- Порода: лабрадор
- Окрас: серебристо-белый
- Глаза: карие, умные
- Характер: оптимистичный, радостный

**Генрих:**
- Порода: немецкая овчарка
- Окрас: чёрно-подпалый
- Глаза: зелёные
- Характер: мудрый, проницательный, с лёгкой печалью

## 💡 Советы по промптам

### Добавляй конкретики:
```
❌ собака → ✅ серебристо-белый лабрадор с карими глазами
❌ на улице → ✅ на солнечной поляне в весеннем лесу
❌ красиво → ✅ в стиле винтажной книжной иллюстрации
```

### Добавляй настроения:
```
❌ весело → ✅ радостно, с беззаботностью щенка
❌ грустно → ✅ с тихой меланхолией и мудростью в глазах
```

### Добавляй освещения:
```
❌ освещение → ✅ мягкий свет золотого часа, тёплые блики
❌ тени → ✅ нежные тени, создающие глубину
```
