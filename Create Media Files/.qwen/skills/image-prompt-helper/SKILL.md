# Image Prompt Helper

## 🎯 Назначение

Этот скилл помогает создавать качественные изображения через платформу https://arena.ai/ru/image, улучшая промпты и выбирая лучшие результаты.

---

## 📋 Рабочий процесс

### Шаг 1: Получение короткого промпта

Пользователь пишет короткий запрос, например:
- "Две собаки на закате"
- "Лабрадор щенок играет"
- "Смельчак и Генрих дружат"

### Шаг 2: Улучшение промпта

**Преобразуй короткий промпт в детализированный, добавив:**

1. **Объекты и персонажи**
   - Кто изображён (порода, окрас, поза, выражение)
   - Количество объектов
   - Взаимное расположение

2. **Окружение и фон**
   - Место действия (лес, поле, дом, улица)
   - Время суток (рассвет, день, закат, ночь)
   - Погода (солнечно, дождь, снег, туман)

3. **Стиль и техника**
   - Художественный стиль (реализм, иллюстрация, акварель, цифровая живопись)
   - Цветовая палитра (тёплая, холодная, пастельная, контрастная)
   - Освещение (мягкое, драматичное, контровое, золотой час)

4. **Настроение и атмосфера**
   - Эмоции (радость, грусть, ностальгия, надежда)
   - Атмосфера (уютная, эпическая, медитативная, динамичная)

5. **Композиция**
   - Ракурс (крупный план, средний, общий план)
   - Перспектива (с уровня глаз, сверху, снизу)
   - Фокус (на переднем плане, глубина резкости)

**Пример улучшения:**

❌ **Коротко:** "Две собаки на закате"

✅ **Детально:** 
```
A heartwarming illustration of two dogs facing each other: a white labrador on the left and a german shepherd on the right, both with joyful expressions and open mouths. Warm sunset lighting with golden rays streaming from corners. Soft peach and orange background. Vector illustration style with clean lines and gentle shadows. Small decorative hearts around text. Mood: friendship, celebration, warmth. Professional digital art, high quality, detailed fur texture.
```

### Шаг 3: Показать пользователю

Представь улучшенный промпт в таком формате:

```
📝 Улучшенный промпт:

[текст промпта на английском]

🎨 Что добавлено:
• Детали персонажей: ...
• Окружение: ...
• Стиль: ...
• Настроение: ...

✅ Одобряете? (да/нет или правки)
```

### Шаг 4: Сохранение промпта

После одобрения пользователем:

1. **Создай файл** в папке `prompts/`
2. **Имя файла:** `YYYY-MM-DD_HH-MM_prompt.txt`
3. **Содержимое файла:**
   ```
   Дата: [дата и время]
   
   Оригинальный промпт:
   [текст]
   
   Улучшенный промпт:
   [текст]
   
   Сгенерировано на: https://arena.ai/ru/image
   ```

### Шаг 5: Генерация изображений

1. **Открой ссылку:** https://arena.ai/ru/image
2. **Пользователь вставляет промпт** в браузере
3. **Генерируются 2 изображения** от разных моделей
4. **Пользователь загружает оба изображения** в чат

### Шаг 6: Анализ изображений

**Сравни оба изображения по 6 критериям:**

| Критерий | Вопросы для анализа |
|----------|---------------------|
| **1. Соответствие промпту** | Насколько точно переданы все элементы промпта? Какие детали упущены? |
| **2. Композиция** | Сбалансирована ли композиция? Есть ли явный фокус? Как расположены объекты? |
| **3. Цветовая палитра** | Гармонируют ли цвета? Соответствует ли настроение палитре? Есть ли цветовые акценты? |
| **4. Детализация** | Проработаны ли мелкие детали (шерсть, глаза, текстуры)? Нет ли размытости? |
| **5. Стиль** | Выдержан ли единый художественный стиль? Выглядит ли профессионально? |
| **6. Эмоциональный отклик** | Какое изображение сильнее "цепляет"? Какое лучше передаёт настроение? |

**Формат анализа:**

```
🖼️ Анализ изображений

=== Изображение 1 ===
✅ Сильные стороны: ...
⚠️ Слабые стороны: ...
Оценка по критериям:
• Соответствие: 8/10
• Композиция: 7/10
• Цвета: 9/10
• Детализация: 8/10
• Стиль: 8/10
• Эмоции: 9/10

=== Изображение 2 ===
✅ Сильные стороны: ...
⚠️ Слабые стороны: ...
Оценка по критериям: ...

=== Рекомендация ===
🏆 Лучшее: Изображение [1/2]

Обоснование:
[Подробное объяснение выбора по ключевым критериям]
```

### Шаг 7: Сохранение результата

Посоветуй пользователю:
- Сохранить лучшее изображение в папку `results/`
- Дать файлу описательное имя (например, `2026-02-28_Smelchak_Genrik_sunset.jpg`)

## 🎨 Принципы улучшения промптов для проекта "Собачья мудрость"

### Цветовая палитра
- **Предпочтительно:** древесные, травянистые, лиловые, фиолетовые тона
- **Избегать:** холодные офисные цвета (серый, стальной, кислотные оттенки)

### Стиль
- **Предпочтительно:** природный ретро-стиль, тёплые иллюстрации, акварель
- **Избегать:** современный минимализм, глянцевый стиль

### Настроение
- **Предпочтительно:** тёплое, эмпатичное, ностальгическое, вдохновляющее
- **Избегать:** мрачное, агрессивное, слишком динамичное

### Персонажи
- **Смельчак:** серебристо-белый лабрадор, умные карие глаза, мускулистое телосложение
- **Генрих:** классический окрас немецкой овчарки, зелёные глаза, густая шерсть

## ⚠️ Важно

- **Не генерируй изображения сам** — ты только улучшаешь промпты и анализируешь результаты
- **Всегда показывай промпт пользователю** перед сохранением
- **Не выбирай изображение без загрузки обоих вариантов**
- **Обосновывай выбор** подробно, по всем 6 критериям
