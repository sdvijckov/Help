# 03_Настройки языка

> Команды для управления языком интерфейса и вывода Qwen Code

---

## Список команд

| Команда | Назначение |
|---------|----------|
| [`/language`](#language) | Просмотр или изменение настроек языка |
| [`/language ui [язык]`](#language-ui-язык) | Установка языка UI интерфейса |
| [`/language output [язык]`](#language-output-язык) | Установка языка вывода LLM |

---

## `/language`

**Назначение:** Показывает текущие настройки языка и позволяет изменить их.

**Когда использовать:**
- Хотите проверить текущие языковые настройки
- Нужно сменить язык интерфейса или вывода
- Работаете в международной команде
- Предпочитаете использовать родной язык

**Что делает:**
- Показывает текущий язык интерфейса (UI)
- Показывает текущий язык вывода модели (output)
- Открывает меню для выбора новых настроек

**Пример:**
```
/language
```

**Результат:** Qwen покажет текущие настройки:
```
Текущие настройки языка:
- Язык интерфейса (UI): ru-RU (Русский)
- Язык вывода (Output): Russian

Выберите действие:
[1] Изменить язык интерфейса
[2] Изменить язык вывода
[3] Отмена
```

---

## `/language ui [язык]`

**Назначение:** Устанавливает язык пользовательского интерфейса Qwen Code.

**Когда использовать:**
- Хотите, чтобы меню и подсказки были на определённом языке
- Работаете с человеком, который предпочитает другой язык
- Нужно переключиться на родной язык для удобства

**Синтаксис:**
```
/language ui <код-языка>
```

**Примеры:**
```
/language ui ru-RU      # Русский
/language ui en-US      # Английский (США)
/language ui zh-CN      # Китайский (упрощённый)
/language ui de-DE      # Немецкий
```

**Доступные языки интерфейса:**

| Код | Язык |
|-----|------|
| `ru-RU` | Русский |
| `en-US` | Английский (США) |
| `zh-CN` | Китайский (упрощённый) |
| `de-DE` | Немецкий |

**Что изменяется:**
- Текст меню и команд
- Подсказки и сообщения об ошибках
- Названия кнопок и элементов интерфейса

**Важно:** Эта команда изменяет только язык интерфейса, но не язык ответов модели.

---

## `/language output [язык]`

**Назначение:** Устанавливает язык, на котором модель (LLM) генерирует ответы.

**Когда использовать:**
- Хотите получать ответы на определённом языке
- Работаете с кодом, но объяснения нужны на родном языке
- Пишете документацию на другом языке
- Общаетесь с международной командой

**Синтаксис:**
```
/language output <название-языка>
```

**Примеры:**
```
/language output Russian      # Русский
/language output English      # Английский
/language output Chinese      # Китайский
/language output German       # Немецкий
```

**Что изменяется:**
- Язык всех ответов Qwen
- Язык объяснений кода
- Язык комментариев и документации
- Язык ошибок и предупреждений

**Важно:** Эта команда не изменяет язык интерфейса, только язык ответов модели.

---

## Комбинированное использование

Вы можете настроить оба языка независимо:

**Пример 1: Русский интерфейс + Английский вывод**
```
/language ui ru-RU
/language output English
```
*Полезно:* Если вам удобнее меню на русском, но код и техническая документация нужны на английском.

**Пример 2: Английский интерфейс + Русский вывод**
```
/language ui en-US
/language output Russian
```
*Полезно:* Если вы привыкли к английскому интерфейсу, но хотите получать объяснения на русском.

**Пример 3: Одинаковый язык (рекомендуется для начинающих)**
```
/language ui ru-RU
/language output Russian
```
*Полезно:* Единый язык для всего снижает когнитивную нагрузку.

## Проверка текущих настроек

Чтобы проверить текущие настройки языка:

```
/language
```

Или посмотрите файл настроек Qwen Code (обычно находится в `~/.qwen/settings.json` или аналогичном месте).

---

## Сброс настроек языка

Чтобы вернуть настройки по умолчанию:

```
/language ui en-US
/language output English
```

Или удалите настройки языка из файла конфигурации.

---

## Советы по использованию

1. **Для русскоязычных пользователей:** Установите оба языка на русский для максимального комфорта.
   ```
   /language ui ru-RU
   /language output Russian
   ```

2. **Для работы с кодом:** Оставьте вывод на английском, если работаете с международной командой.
   ```
   /language ui ru-RU
   /language output English
   ```

3. **Для изучения английского:** Используйте английский интерфейс с русским выводом для постепенного погружения.
   ```
   /language ui en-US
   /language output Russian
   ```

4. **Проверка перед важной работой:** Убедитесь, что язык настроен правильно перед началом важной задачи.
   ```
   /language
   ```
