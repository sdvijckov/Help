## 📋 Краткий справочник

### Slash-команды (`/`)

#### Управление сессией
| Команда | Назначение |
|---------|----------|
| `/init` | Анализ проекта и создание контекста |
| `/summary` | Сводка текущей сессии |
| `/compress` | Сжатие истории для экономии токенов |
| `/resume` | Возобновление предыдущей сессии |
| `/restore` | Восстановление файлов |

#### Интерфейс
| Команда | Назначение |
|---------|----------|
| `/clear` | Очистка экрана |
| `/theme` | Смена темы |
| `/vim` | Режим Vim |
| `/directory` | Управление папками |
| `/editor` | Выбор редактора |

#### Язык
| Команда | Назначение |
|---------|----------|
| `/language` | Настройки языка |
| `/language ui` | Язык интерфейса |
| `/language output` | Язык вывода |

#### Инструменты
| Команда | Назначение |
|---------|----------|
| `/mcp` | MCP серверы |
| `/tools` | Доступные инструменты |
| `/skills` | Навыки |
| `/model` | Переключение модели |
| `/extensions` | Расширения |
| `/memory` | Память ИИ |

#### Режимы подтверждения
| Команда | Уровень |
|---------|---------|
| `/approval-mode plan` | 🔒 Только анализ |
| `/approval-mode default` | 🛡️ Стандартный |
| `/approval-mode auto-edit` | ⚡ Авто-редактирование |
| `/approval-mode yolo` | 🚀 Полное авто |

#### Информация
| Команда | Назначение |
|---------|----------|
| `/help` | Справка |
| `/about` | О версии |
| `/stats` | Статистика |
| `/settings` | Настройки |
| `/auth` | Аутентификация |
| `/bug` | Сообщение об ошибке |
| `/copy` | Копировать вывод |
| `/quit` | Выход |

---

### File-команды (`@`)

| Синтаксис | Назначение |
|-----------|----------|
| `@<файл>` | Вставить содержимое файла |
| `@<директория>/` | Прочитать директорию |

**Пример:**
```
Объясни код: @src/main.py
```

---

### Shell-команды (`!`)

| Синтаксис | Назначение |
|-----------|----------|
| `!<команда>` | Выполнить команду ОС |

**Примеры:**
```
!git status
!npm install
!python --version
```

---

### Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl+L` | Очистить экран |
| `Ctrl+C ×2` | Выйти |
| `Ctrl+T` | Описание инструментов |
| `Ctrl+Z` | Отменить |
| `↑/↓` | История команд |
| `Tab` | Автодополнение |

---

### Субагенты (`/agents`)

| Команда | Назначение |
|---------|----------|
| `/agents create` | Создание субагента |
| `/agents manage` | Управление субагентами |

**Примеры субагентов:**
- `testing-expert` — написание тестов
- `documentation-writer` — документация
- `code-reviewer` — ревью кода
- `python-expert` — Python разработка

## 💡 Советы для начинающих

1. **Начните с `/init`** — создайте базовый контекст проекта
2. **Используйте `@` для файлов** — быстрее, чем копировать код
3. **Проверяйте `/stats`** — следите за использованием токенов
4. **Настройте `/approval-mode`** — выберите комфортный уровень безопасности
5. **Создавайте свои команды** — автоматизируйте частые задачи
6. **Используйте субагентов** — делегируйте специализированные задачи
