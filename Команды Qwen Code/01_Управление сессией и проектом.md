# 01_Управление сессией и проектом

> Команды для управления сессиями, историей разговора и состоянием проекта

---

## Список команд

| Команда | Назначение |
|---------|----------|
| [`/init`](#init) | Анализ проекта и создание контекстного файла |
| [`/summary`](#summary) | Создание сводки текущей сессии |
| [`/compress`](#compress) | Сжатие истории чата для экономии токенов |
| [`/resume`](#resume) | Возобновление предыдущей сессии |
| [`/restore`](#restore) | Восстановление файлов до предыдущего состояния |

---

## `/init`

**Назначение:** Анализирует текущую директорию проекта и создаёт начальный контекстный файл для Qwen Code.

**Когда использовать:**
- Начинаете работу с новым проектом
- Хотите, чтобы Qwen понял структуру вашего проекта
- Нужно создать базовую документацию автоматически

**Что делает:**
- Сканирует файлы и папки в текущей директории
- Определяет тип проекта (Python, JavaScript, etc.)
- Создаёт файл с описанием структуры и назначения проекта
- Сохраняет контекст для будущих сессий

**Пример:**
```
/init
```

**Результат:** Qwen создаст файл (обычно `.qwen/PROJECT_CONTEXT.md` или подобный) с описанием вашего проекта.

---

## `/summary`

**Назначение:** Создаёт сводку текущего разговора на основе всей истории сессии.

**Когда использовать:**
- Длинная сессия, нужно зафиксировать прогресс
- Перед перерывом в работе
- Хотите сохранить ключевые решения и выводы
- Планируете использовать `/compress`

**Что делает:**
- Анализирует всю историю переписки
- Выделяет ключевые моменты, решения, изменения в коде
- Генерирует структурированную сводку
- Сохраняет для будущего использования

**Пример:**
```
/summary
```

**Результат:** Получите текстовую сводку вида:
```
## Сводка сессии

### Выполненные задачи:
- Создан модуль авторизации
- Исправлена ошибка валидации email

### Изменённые файлы:
- src/auth.py
- src/validators.py

### Открытые вопросы:
- Требуется тестирование интеграции с OAuth
```

## `/compress`

**Назначение:** Заменяет длинную историю чата на сжатую сводку для экономии токенов.

**Когда использовать:**
- Сессия стала очень длинной
- Заканчивается лимит токенов контекста
- Хотите продолжить работу, но начать «с чистого листа»
- После завершения крупного этапа работы

**Что делает:**
- Создаёт сводку текущей сессии (как `/summary`)
- Заменяет всю историю на эту сводку
- Освобождает место в контекстном окне
- Сохраняет суть для продолжения работы

**Пример:**
```
/compress
```

**Важно:** После сжатия нельзя вернуться к деталям ранних сообщений. Используйте `/summary` перед `/compress`, чтобы сохранить полную версию отдельно.

---

## `/resume`

**Назначение:** Возобновляет предыдущую сессию разговора.

**Когда использовать:**
- Продолжаете работу после перерыва
- Нужно восстановить контекст вчерашней сессии
- Работаете над долгосрочным проектом поэтапно

**Что делает:**
- Находит последнюю сохранённую сессию
- Восстанавливает историю переписки
- Возвращает контекст проекта

**Пример:**
```
/resume
```

**Примечание:** Сессии обычно сохраняются автоматически. Если сессий несколько, Qwen может предложить выбрать нужную.

---

## `/restore`

**Назначение:** Восстанавливает файлы проекта в состояние до выполнения инструментов Qwen.

**Когда использовать:**
- Qwen внёс нежелательные изменения в код
- Хотите откатить последние правки
- Экспериментировали с разными вариантами решения

**Синтаксис:**
```
/restore              # Показать список доступных точек восстановления
/restore <ID>         # Восстановить конкретную точку
```

**Пример:**
```
/restore
# Qwen покажет список:
# [1] 14:30 - Изменения в auth.py
# [2] 14:45 - Добавлен валидатор email

/restore 1
# Файл auth.py будет восстановлен
```

**Важно:** Эта команда работает только с изменениями, внесёнными инструментами Qwen в текущей сессии.
