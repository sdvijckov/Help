# 06_Информация и настройки

> Команды для получения информации, настройки и управления Qwen Code

---

## Список команд

| Команда | Назначение |
|---------|----------|
| [`/help`](#help) | Показать справочную информацию |
| [`/about`](#about) | Показать информацию о версии |
| [`/stats`](#stats) | Показать детальную статистику сессии |
| [`/settings`](#settings) | Открыть редактор настроек |
| [`/auth`](#auth) | Изменить метод аутентификации |
| [`/bug`](#bug) | Отправить сообщение о проблеме |
| [`/copy`](#copy) | Скопировать последний вывод в буфер обмена |
| [`/quit`](#quit) | Выйти из Qwen Code |

---

## `/help`

**Назначение:** Показывает справочную информацию по доступным командам.

**Когда использовать:**
- Забыли синтаксис команды
- Хотите увидеть список всех команд
- Начинаете работу с Qwen Code
- Нужна быстрая подсказка

**Что делает:**
- Показывает список всех доступных команд
- Отображает краткое описание каждой команды
- Предоставляет примеры использования

**Пример:**
```
/help
```

**Результат:**
```
Доступные команды:

/init          — Анализ проекта и создание контекста
/summary       — Создание сводки сессии
/compress      — Сжатие истории чата
/clear         — Очистить экран
/theme         — Сменить тему
/language      — Настройки языка
/tools         — Показать инструменты
/model         — Переключить модель
/stats         — Показать статистику
/settings      — Открыть настройки
/quit          — Выйти из Qwen Code

Введите /help <команда> для подробной информации.
```

**Поиск по командам:**
```
/help language
```
Покажет подробную информацию о команде `/language`.

**Альтернатива:** `/?` — короткий аналог команды `/help`.

## `/about`

**Назначение:** Показывает информацию о версии Qwen Code и установленных компонентах.

**Когда использовать:**
- Нужно узнать версию для отчёта об ошибке
- Хотите проверить, установлена ли последняя версия
- Нужна информация для поддержки
- Любопытно узнать о сборке

**Что делает:**
- Показывает номер версии
- Отображает номер сборки
- Показывает информацию о платформе
- Перечисляет активные компоненты

```
/about
```

**Результат:**

## `/stats`

**Назначение:** Показывает детальную статистику использования в текущей сессии.

**Когда использовать:**
- Хотите отслеживать использование токенов
- Нужно оптимизировать расход токенов
- Интересует производительность сессии
- Планируете использовать `/compress`

**Что делает:**
- Показывает использование токенов
- Отображает длительность сессии
- Считает количество запросов
- Показывает использование инструментов

**Пример:**
```
/stats
```

**Результат:**
```
📊 Статистика сессии

⏱️ Время сессии: 2ч 34мин
💬 Запросов: 127
🔧 Использовано инструментов: 45

📈 Использование токенов:
  Входные токены:  45,230
  Выходные токены: 23,450
  Всего:           68,680 / 100,000 (68.7%)

📁 Изменено файлов: 12
📝 Создано файлов: 3
```

**Почему это важно:**
- **Токены** — ограниченный ресурс, влияет на стоимость

- **Файлы** — отслеживает изменения в проекте

**Совет:** Если использование токенов > 80%, рассмотрите `/compress`.

## `/settings`

**Назначение:** Открывает редактор настроек Qwen Code.

**Когда использовать:**
- Хотите изменить настройки по умолчанию
- Нужно настроить поведение Qwen
- Хотите изменить язык или тему
- Нужен доступ к конфигурационному файлу

**Что делает:**
- Открывает файл настроек в редакторе
- Позволяет изменить параметры
- Применяет изменения после сохранения

**Пример:**
```
/settings
```

**Результат:** Открывается файл настроек (обычно `~/.qwen/settings.json`):

```json
{
  "theme": "dark",
  "language": {
    "ui": "ru-RU",
    "output": "Russian"
  },
  "approvalMode": "default",
  "model": "Qwen-Plus",
  "autoSave": true,
  "maxTokens": 100000
}
```

**Основные настройки:**

| Параметр | Описание | Значения |
|----------|----------|----------|
| `theme` | Визуальная тема | `dark`, `light`, `system` |
| `language.ui` | Язык интерфейса | `ru-RU`, `en-US`, `zh-CN`, `de-DE` |
| `language.output` | Язык вывода | `Russian`, `English`, `Chinese`, `German` |
| `approvalMode` | Режим подтверждения | `plan`, `default`, `auto-edit`, `yolo` |
| `model` | Модель по умолчанию | `Qwen-Max`, `Qwen-Plus`, `Qwen-Turbo` |
| `autoSave` | Автосохранение сессий | `true`, `false` |
| `maxTokens` | Лимит токенов на сессию | число |

**Совет:** После изменения настроек перезапустите Qwen Code для применения.

## `/auth`

**Назначение:** Изменяет метод аутентификации для доступа к сервисам Qwen.

**Когда использовать:**
- Нужно сменить аккаунт
- Хотите использовать другой API-ключ
- Проблемы с текущей аутентификацией
- Переключаетесь между личным и рабочим аккаунтом

**Что делает:**
- Показывает текущий метод аутентификации
- Позволяет выбрать новый метод
- Обновляет учётные данные

**Пример:**
```
/auth
```

**Результат:**
```
Текущая аутентификация: Google OAuth (user@example.com)

Выберите метод аутентификации:

[1] Google OAuth
[2] API Key
[3] Выйти из аккаунта
[4] Отмена

Ваш выбор:
```

**Методы аутентификации:**

1. **Google OAuth** — вход через аккаунт Google (рекомендуется)
   - Удобно
   - Безопасно
   - Автоматическое обновление токена

2. **API Key** — использование API-ключа
   - Для автоматизации
   - Для серверов
   - Требует ручного обновления

3. **Выйти из аккаунта** — сброс аутентификации
   - Для смены пользователя
   - Для безопасности

**Важно:** После смены аутентификации может потребоваться перезапуск сессии.

## `/bug`

**Назначение:** Отправляет сообщение о проблеме разработчикам Qwen Code.

**Когда использовать:**
- Обнаружили ошибку в работе
- Нашли баг в интерфейсе
- Столкнулись с некорректным поведением
- Хотите предложить улучшение

**Что делает:**
- Собирает информацию о сессии
- Создаёт отчёт об ошибке
- Отправляет разработчикам
- Создаёт issue на GitHub (если возможно)

**Пример:**
```
/bug
```

**Результат:**
```
📋 Отчёт об ошибке

Опишите проблему:
> [Здесь вы вводите описание]

---

Автоматически собранная информация:

Версия: 1.5.2
Платформа: Windows 11 (win32)
Сессия: 2ч 34мин
Запросов: 127

Последние команды:
- /stats
- /tools
- /model

[1] Отправить отчёт
[2] Добавить скриншот
[3] Отмена
```

**Что включается в отчёт:**
- Версия Qwen Code
- Информация о платформе
- История команд
- Логи ошибок
- Использование токенов

**Альтернатива:** Можно создать issue напрямую на GitHub:
https://github.com/QwenLM/qwen-code/issues

## `/copy`

**Назначение:** Копирует последнее содержимое вывода в буфер обмена.

**Когда использовать:**
- Хотите сохранить результат работы
- Нужно вставить код в другое место
- Хотите поделиться выводом
- Нужна быстрая копия без выделения

**Что делает:**
- Копирует последний ответ Qwen
- Сохраняет в буфер обмена системы
- Уведомляет об успешном копировании

**Пример:**
```
/copy
```

**Результат:**
```
✅ Скопировано в буфер обмена!
```

**Что копируется:**
- Последний текстовый ответ
- Код из последнего сообщения
- Таблицы и структурированные данные

**Важно:** Команда копирует только текст, не изображения или файлы.

## `/quit` (или `/exit`)

**Назначение:** Немедленно завершает сессию и выходит из Qwen Code.

**Когда использовать:**
- Завершили работу
- Нужно быстро выйти
- Закончили сессию
- Хотите освободить ресурсы

**Что делает:**
- Сохраняет текущую сессию (если включено автосохранение)
- Закрывает соединение с сервером
- Завершает процесс Qwen Code
- Возвращает в командную строку

**Пример:**
```
/quit
```

**Или:**
```
/exit
```

**Результат:**
```
💾 Сессия сохранена.
👋 До встречи!

D:\Projects python\Help>
```

**Важно:**
- Перед выходом сессия автоматически сохраняется
- Можно возобновить через `/resume`
- Убедитесь, что все изменения сохранены в файлах

**Горячая клавиша:** `Ctrl+C` (дважды) — альтернативный способ выхода.

## Рабочий процесс с командами информации

**Типичная сессия:**

```
# Начало работы
/about              # Проверить версию
/stats              # Проверить лимиты

# В процессе
/stats              # Мониторинг токенов
/help <команда>     # Если забыли синтаксис

# При проблеме
/bug                # Сообщить об ошибке

# Завершение
/stats              # Итоговая статистика
/copy               # Скопировать результат
/quit               # Выйти
```

---

## Советы по использованию

1. **Проверяйте `/stats` регулярно** — следите за расходом токенов.

2. **Используйте `/help <команда>`** — быстрая справка по командам.

3. **Копируйте важное через `/copy`** — не теряйте результаты.

4. **Сообщайте о багах через `/bug`** — помогите улучшить Qwen.

5. **Сохраняйте версию из `/about`** — пригодится при обращении в поддержку.

6. **Настройте `/settings` под себя** — персонализируйте опыт.
