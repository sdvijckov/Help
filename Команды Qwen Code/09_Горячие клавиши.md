# 09_Горячие клавиши

> Быстрые комбинации клавиш для эффективной работы с Qwen Code

---

## Что такое горячие клавиши?

Горячие клавиши — это комбинации клавиш, которые позволяют быстро выполнять команды без ввода текста.

---

## Основные горячие клавиши

### Навигация и управление

| Комбинация | Действие | Эквивалент команды |
|------------|----------|-------------------|
| `Ctrl + L` | Очистить экран | `/clear` |
| `Ctrl + C` (×2) | Подтверждение выхода | `/quit` |
| `Ctrl + D` | Завершить ввод (EOF) | — |
| `Ctrl + Z` | Отменить ввод (Windows) | — |

---

### Редактирование текста

| Комбинация | Действие |
|------------|----------|
| `Ctrl + Z` | Отменить последнее действие |
| `Ctrl + Shift + Z` | Повторить отменённое действие |
| `Ctrl + X` | Выделить текст |
| `Ctrl + C` | Копировать текст |
| `Ctrl + V` | Вставить текст |
| `Ctrl + A` | Выделить весь текст |
| `Ctrl + F` | Найти текст |
| `Ctrl + H` | Найти и заменить |

---

### Работа с инструментами

| Комбинация | Действие |
|------------|----------|
| `Ctrl + T` | Переключить описание инструментов |
| `Ctrl + Shift + P` | Палитра команд (если доступно) |
| `Tab` | Автодополнение команды |
| `↑` / `↓` | Навигация по истории команд |

---

## Подробное описание

### `Ctrl + L` — Очистить экран

**Назначение:** Быстро очищает терминал от видимого текста.

**Когда использовать:**
- Экран заполнился выводом
- Начинаете новый этап работы
- Хотите «освежить» вид

**Эквивалент:** `/clear`

**Пример:**
```
[Много текста на экране]

Ctrl + L

[Экран очищен, готов к новой работе]
```

**Важно:** История сессии сохраняется, очищается только визуальная часть.

---

### `Ctrl + C` (×2) — Подтверждение выхода

**Назначение:** Безопасный выход из Qwen Code.

**Как работает:**
1. Первое нажатие `Ctrl + C` — прерывает текущую операцию
2. Второе нажатие `Ctrl + C` — подтверждает выход

**Когда использовать:**
- Нужно быстро завершить сессию
- Прервать долгую операцию
- Экстренный выход

**Эквивалент:** `/quit` или `/exit`

**Пример:**
```
Qwen: [Генерирует код...]

Ctrl + C
[Генерация прервана]

Ctrl + C
💾 Сессия сохранена.
👋 До встречи!
```

**Важно:** Сессия автоматически сохраняется перед выходом.

---

### `Ctrl + T` — Переключить описание инструментов

**Назначение:** Показывает или скрывает подробное описание доступных инструментов.

**Когда использовать:**
- Хотите увидеть детали инструментов
- Нужно скрыть описание для экономии места
- Изучаете возможности Qwen

**Пример:**
```
Ctrl + T

[Показано подробное описание всех инструментов]

Ctrl + T

[Описание скрыто, только названия]
```

---

### `Ctrl + Z` — Отменить действие

**Назначение:** Отменяет последнее действие в поле ввода.

**Когда использовать:**
- Случайно удалили текст
- Хотите вернуть предыдущее состояние
- Ошиблись при вводе

**Windows:**
- `Ctrl + Z` — отмена
- `Ctrl + Y` или `Ctrl + Shift + Z` — повтор

**Linux/Mac:**
- `Ctrl + Z` — отмена
- `Ctrl + Shift + Z` — повтор

---

### `↑` / `↓` — История команд

**Назначение:** Навигация по ранее введённым командам.

**Как использовать:**
- `↑` — предыдущая команда
- `↓` — следующая команда

**Пример:**
```
[Ввели: /stats]
[Ввели: /tools]

↑  ← Покажет: /tools
↑  ← Покажет: /stats
↓  ← Покажет: /tools
↓  ← Пустая строка (готов к вводу)
```

**Совет:** Можно редактировать команду из истории перед отправкой.

---

### `Tab` — Автодополнение

**Назначение:** Автоматически дополняет команды и пути.

**Когда использовать:**
- Забыли полное название команды
- Хотите быстрее ввести команду
- Работаете с длинными путями

**Пример:**
```
/he[TAB]     → /help
/sum[TAB]    → /summary
@src/main[TAB] → @src/main.py
```

**Продвинутое использование:**
- Двойной `Tab` — показать все варианты
- `Tab` после `@` — автодополнение путей к файлам

## Советы по использованию

### ✅ Лучшие практики

1. **Запомните основные комбинации:**
   - `Ctrl + L` — очистить экран
   - `Ctrl + C` (×2) — выйти
   - `↑` / `↓` — история команд

2. **Используйте `Tab` для автодополнения** — экономит время.

3. **Комбинируйте с командами:**
   ```
   /he[Tab]  →  /help
   ```

4. **Используйте историю команд** — не вводите одно и то же дважды.

5. **Настройте под себя** — некоторые комбинации можно переназначить.

---

### ⚡ Продуктивность

**Быстрый рабочий процесс:**

```
1. /init [Enter]        # Инициализация
2. [Ждём завершения]
3. Ctrl + L             # Очистить экран
4. @src/ [Enter]        # Добавить файлы
5. [Вопрос к Qwen]
6. ↑ [Enter]            # Повторить вопрос
7. Ctrl + C (×2)        # Выйти
```

**Экономия времени:**
- `Tab` вместо полного ввода команд
- `↑` вместо повторного ввода
- `Ctrl + L` вместо `/clear`

---

### 🔧 Настройка горячих клавиш

Некоторые версии Qwen Code позволяют настраивать горячие клавиши.

**Файл настроек:** `~/.qwen/keybindings.json`

**Пример:**
```json
{
  "clearScreen": "Ctrl+L",
  "quit": "Ctrl+Q",
  "historyUp": "Up",
  "historyDown": "Down",
  "autoComplete": "Tab"
}
```

## Решение проблем

### Горячие клавиши не работают

**Возможные причины:**
1. Конфликт с системными комбинациями
2. Терминал перехватывает нажатия
3. Неподдерживаемая комбинация

**Решение:**
1. Проверьте настройки терминала
2. Используйте альтернативные комбинации
3. Вводите команды вручную

---

### `Ctrl + C` не прерывает

**Проблема:** Команда выполняется и не прерывается.

**Решение:**
1. Подождите завершения операции
2. Попробуйте `Ctrl + Z` (поставить на паузу)
3. Закройте терминал (крайняя мера)

---

### `Tab` не дополняет

**Проблема:** Автодополнение не работает.

**Решение:**
1. Убедитесь, что начали вводить команду
2. Проверьте, существует ли файл/команда
3. Попробуйте двойной `Tab` для списка вариантов
